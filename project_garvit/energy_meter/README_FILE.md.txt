energy meter
In this the current sensor and voltage divider circuit takes the voltage as an input and with the help of the Arduino we measure the values of the current and voltage and with that we measure the power and energy 

Components required – 
•	Arduino Uno 
•	Temperature Sensor (LM35 / DHT11) 
•	Breadboard / PCB 
•	Connecting Wires

Working – 
•	The temperature sensor (LM35/DHT11) detects ambient temperature. 
•	LM35 outputs an analog voltage proportional to temperature. 
•	Arduino reads this value using an analog pin. 
•	The value is converted into temperature (°C). 
•	The result is displayed on LCD or Serial Monitor.
