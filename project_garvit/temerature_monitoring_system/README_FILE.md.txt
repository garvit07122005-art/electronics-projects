temperature monitoring system - 

in this the temp sensor takes the input and the potentiometer is used to set the threshold , if the temp goes above the limit set with the potentiometer then the alert will be on 


Components required – 
•	Arduino Uno 
•	Temperature Sensor (LM35 / DHT11) 
•	Breadboard / PCB 
•	Connecting Wires

Working – 
•	The temperature sensor (LM35/DHT11) detects ambient temperature. 
•	LM35 outputs an analog voltage proportional to temperature. 
•	Arduino reads this value using an analog pin. 
•	The value is converted into temperature (°C). 
•	The result is displayed on LCD or Serial Monitor.
