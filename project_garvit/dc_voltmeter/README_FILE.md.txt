DC_voltmeter

it measures the dc voltage with the help of the voltage divider circuit  
 
Components Required – 
•	Arduino Uno 
•	Coil (Inductor) 
•	Capacitor 
•	Buzzer 
•	Resistors 
•	Breadboard / PCB 
•	Connecting Wires

Working – 
•	A coil is used to generate a magnetic field when current flows through it. 
•	When a metal object comes near the coil, it disturbs the magnetic field. 
•	This change affects the circuit parameters (inductance/frequency). 
•	Arduino detects this variation and triggers a buzzer as an alert.
